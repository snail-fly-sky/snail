# Snail Tools

一个 Vue3 + JavaScript + Element Plus 的个人工具入口项目，包含简历工作台、JSON 处理和时间转换。

## 功能

- 首页工具入口
- 多份简历管理
- 基础信息、技能、工作经历、项目经历、教育经历编辑
- 浏览器 `localStorage` 自动保存
- 当前简历导出为 PDF
- 当前简历导出为 Word 可打开的 `.doc` 文件
- JSON 格式化与压缩
- 时间戳与本地时间转换
- Vue Router 管理页面与简历详情地址
- Pinia 管理简历数据与编辑动作

## 运行

```bash
npm install
npm run dev
```
